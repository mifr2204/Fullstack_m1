OM BOOTSTRAP

Bootstrap är ett CSS-ramverk som finns att hitta på https://getbootstrap.com/ Man kan integrera bootstrap på flera olika sätt.
Man kan ladda ner och installera det, använda CDN eller använda en package manager.
Jag valde att använda CDN då man bara behöver använda en länk i html koden för att få tillgång till biblioteket.
Bootstrap räknas som ett av de större traditionella ramverken

MIN ERFARENHET AV BOOTSTRAP

Detta är det första ramverket jag någonsin använt och jag ÄLSKAR det.
Det gick så mycket fortare att skapa de olika komponenterna och jag tycker att det finns mycket möjligheter för att kunna justera utseende osv själv.
Innan jag testade detta trodde jag att man fick färdiga komponenter som var "fixerade" och inte gick justera, vilket då inte var fallet.
Jag har ännu inte hittat något jag tycker är negativt annat än att jag har en inlärningskurva och har lite svårt att hitta de olika klasserna jag behöver.
